# 🎓 AI-Powered Personalized Learning System

An intelligent adaptive learning platform that personalizes quizzes and tracks student performance using Machine Learning.

---

## 📌 Project Overview

This project aims to solve the problem of one-size-fits-all education by:

- Tracking student performance
- Identifying weak and strong topics
- Recommending personalized quizzes
- Adapting difficulty level dynamically

---

## 🧠 Tech Stack

- **Backend:** FastAPI
- **Database:** PostgreSQL (Supabase)
- **Machine Learning:** Python (scikit-learn)
- **Frontend:** (To be implemented)
- **Version Control:** Git & GitHub
